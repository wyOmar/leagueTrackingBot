# FeedWatch

Welcome to the official documentation and legal hub for the **FeedWatch** Discord Bot.

FeedWatch is a 100% free, multi-tenant Discord bot that connects to the Riot Games API to track League of Legends matches, provide live dashboards, and announce post-match statistics for your server.
